# Social Media Network

A social media network that allows developers to create profiles and share content.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. See deployment for notes on how to deploy the project on a live system.

### Prerequisites

What things you need to install the software and how to install them

```
1. Download the latest version of Node.js
2. Create a mlab account and create an AWS Sandbox database for free.
3. Download Microsoft Visual Studio Code (Optional)
```

### Installing

A step by step series of examples that tell you how to get a development env running

1. Create a folder and open the terminal or git bash from there and create a node project by running the following command:

```
npm init
```

2. Download required dependencies using the following command:

```
npm i express mongoose passport passport-jwt jsonwebtoken body-parser bcryptjs validator
```

3. Download required dev dependency

```
npm i -D nodemon
```

4. fork
